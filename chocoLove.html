Choco Love

A small puzzle game written in 3 dayss for the LÖVE 2021 game jam with the given theme of 'Chocolate'. It was packaged using the LÖVE Web Builder, and the source code is available on GitHub.
Controls
arrows/wasd: move
r: reset level
